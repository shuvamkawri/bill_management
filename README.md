# Bill Management App

A simple and user-friendly Bill Management App designed to help users manage, track, and organize their bills and expenses efficiently. This app provides an intuitive interface for adding, viewing, editing, and tracking bills, ensuring users never miss a payment and can easily manage their financial commitments.

## Features

- **Add Bills**: Add new bills with details like name, due date, amount, and payment status.
- **View Bills**: View all your bills in a list or calendar view.
- **Edit Bills**: Update details for any bill such as payment status, amount, or due date.
- **Search and Filter**: Search for specific bills or filter by categories, due dates, or payment status.
- **Recurring Bills**: Set up recurring bills with customizable intervals (e.g., weekly, monthly, etc.).
- **Analytics**: View your total outstanding bills and monthly expenditure summary.
-**Print**: User print or save the bill 


## Technologies Used

- **Flutter**: Cross-platform mobile development framework.
- **SQLite**: Local database to store bill data. 
- **Dart**: Programming language for app development.
- **Provider**: State management for better performance and handling app states.
